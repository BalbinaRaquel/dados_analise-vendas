# Análise de Dados: Vendas 

# Sobre o Projeto

Aplicação de análise de dados de vendas em linguagem Python com Pandas e Plotly. 
Proposta de projeto do curso da [Empowerdata Treinamentos](https://www.empowerdata.com.br/). 


# Tecnologias utilizadas

- python
- pandas
- plotly_express
